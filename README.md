# Dashboards-de-Usuarios
Este projeto é um sistema simples de gestão acadêmica desenvolvido em Python. Ele calcula automaticamente a média final de cada aluno, determina sua situação (Aprovado, Reprovado, Excelência, etc.) e organiza os dados em ficheiros.  Além disso, gera automaticamente um quadro de honra com os alunos que obtiveram melhor desempenho.
